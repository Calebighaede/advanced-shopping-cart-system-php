# Advanced Shopping Cart System (PHP)

## Overview
This project is a full-stack web-based shopping cart system developed using PHP and MySQL. It simulates a basic e-commerce platform where users can browse products, add items to a cart, and manage their purchases.

## Features
- User registration and login system
- Product listing and management
- Add to cart functionality
- Update and remove items from cart
- Checkout system
- Session-based cart handling
- Database integration using MySQL

## Technologies Used
- PHP (Backend)
- HTML, CSS, JavaScript (Frontend)
- MySQL (Database)

## System Architecture
- Frontend: User interface for browsing and shopping
- Backend: PHP handles business logic and sessions
- Database: MySQL stores users, products, and orders

## How to Run
1. Install XAMPP or WAMP
2. Import the SQL file into MySQL
3. Place project folder in `htdocs`
4. Start Apache and MySQL
5. Open browser and go to:
   http://localhost/project-folder
